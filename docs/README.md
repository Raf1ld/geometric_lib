# Math formulas

## Area

### Circle: S = πR² 

#### Function: 
        (radius) >> circleArea

#### Parameters: 
        (long double) radius

#### Return Value: 
        (long double) circleArea

### Rectangle: S = ab

#### Function: 
        (side1, side2) >> rectangleArea

#### Parameters: 
        (long double) side1 
        (long double) side2 

#### Return Value: 
        (long double) rectangleArea 

### Triendle: S = ah/2

#### Function: 
        (side, height) >> triengleArea

#### Parameters: 
        (long double) side 
        (long double) height 

#### Return Value: 
        (long double) triengleArea   

### Square: S = a²

#### Function: 
        (side) >> squeraArea

#### Parameters: 
        (long double) side

#### Return Value: 
        (long double) squeraArea 

## Perimeter

#### Function:
        (radius) >> circlePerimeter

        (long double) radius

#### Return Value:

        (side1, side2) >> rectanglePerimeter

#### Parameters:

#### Return Value:
        (long double) rectanglePerimeter

        (long double) trienglePerimeter 

### Square: P = 4a

#### Return Value:
