# Math formulas

## Area

### Circle: S = πR² 

#### Function: 
        (radius) >> circleArea

#### Parameters: 
        (long double) radius

#### Return Value: 
        (long double) circleArea

### Rectangle: S = ab

#### Function: 
        (side1, side2) >> rectangleArea

#### Parameters: 
        (long double) side1 
        (long double) side2 

#### Return Value: 
        (long double) rectangleArea 

### Square: S = a²

#### Function: 
        (side) >> squeraArea

#### Parameters: 
        (long double) side

#### Return Value: 
        (long double) squeraArea 

## Perimeter

### Circle: P = 2πR

#### Function:
        (radius) >> circlePerimeter

#### Parameters: 
        (long double) radius

#### Return Value: 
        (long double) circlePerimeter 

### Rectangle: P = 2a + 2b

#### Function:
        (side1, side2) >> rectanglePerimeter

#### Parameters: 
        (long double) side1
        (long double) side2 

#### Return Value:
        (long double) rectanglePerimeter 

### Square: P = 4a

#### Function:
        (side) >> squeraPerimeter

#### Parameters: 
        (long double) side

#### Return Value: 
        (long double) squeraPerimeter
